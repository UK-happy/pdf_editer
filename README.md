# PDF Editor

ブラウザ上で動作するローカルPDF編集ツールです。
サーバーへのアップロードを行わず、すべてブラウザ内で処理が完結します。

## 🚀 起動方法

### 前提条件

- Node.js がインストールされていること

### 手順

```bash
# 依存パッケージのインストール（CMap・フォントデータも自動コピーされます）
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで `http://localhost:5173/` を開いてください。

## 🛠 機能

| 機能               | 説明                                                  |
| ------------------ | ----------------------------------------------------- |
| **Import**         | PDFファイルを読み込み、ページをプレビュー表示         |
| **ページ選択**     | カードをクリックして選択（シアン色でハイライト）      |
| **Insert**         | 別のPDFを選択ページの次に挿入（未選択時は末尾に追加） |
| **ページ削除**     | 各ページのゴミ箱アイコンで削除                        |
| **ページ並び替え** | ドラッグ&ドロップで順番を変更                         |
| **Export**         | 編集後のPDFをダウンロード                             |

## 技術スタック

- **Vite** — ビルドツール・開発サーバー
- **pdf-lib** — PDF操作（ページ削除・挿入・並び替え・生成）
- **pdfjs-dist** — PDFプレビュー描画（CMap対応で日本語OK）
